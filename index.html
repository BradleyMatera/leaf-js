About This Demo

This repository showcases what the leaf-js library is designed to do - providing a performant 3D graphics rendering solution using cutting-edge WebGPU technology. WebGPU is the next-generation web standard for graphics and compute, offering significant performance improvements over WebGL.

Purpose: This demo exists to showcase the code and associated libraries, as described in their documentation and source references. No original development or novel feature implementation has been performed here—this is strictly a demonstration for educational or illustrative purposes.

Browser Compatibility

This demo requires a browser with WebGPU support. WebGPU is currently supported in:

  • Chrome/Edge 113+ (stable)
  • Firefox Nightly (with flags enabled)
  • Safari Technology Preview (experimental)

If you see an error message in the canvas above, your browser may not support WebGPU yet. Please try using the latest version of Chrome or Edge.
